# Visual Cookbook

Welcome to the **Visual Cookbook**, a collection of curated prompts, recipes, and guides for creating high-quality visual content using generative AI models.

## Contents

This repository currently focuses on Google's generative media models:

- **[Imagen Prompts](imagen-prompts.MD)**: A collection of specific prompt recipes for generating various image styles with Imagen.
  - *Examples include:* Professional headshots, 3D character figurines, 80s yearbook photos, vertical portraits, and photo restoration.

- **[Veo Prompts](veo-prompts.MD)**: A comprehensive guide and "meta-prompt" for crafting cinematic video generation prompts with Veo.
  - *Features:* A structured process for deconstructing requests, building immersive worlds, and specifying camera/cinematography details to ensure high visual fidelity.

## Usage

These documents are designed to be used as:
1.  **Direct Prompts**: Copy specific prompt structures to generate images/videos.
2.  **System Instructions**: Use the guides (especially for Veo) to instruct AI assistants on how to refine simple user requests into detailed, professional prompts.
3.  **Inspiration**: Browse different styles and techniques to expand your creative toolkit.
